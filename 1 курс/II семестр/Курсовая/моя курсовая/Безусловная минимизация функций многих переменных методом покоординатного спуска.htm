Многомерная оптимизация. Метод покоординатного спуска.

 

Для решения поставленной задачи методом покоординатного спуска в качестве вспомогательного будем использовать метод Фибоначчи(для нахождения минимума функции m-переменных, если все переменные, кроме одной, фиксированы). Параметры функции :

a,b - левая и правая границы отрезка, на котором ищется минимум;

f - функция m-переменных (m>=2);

v - вектор приближённых значений координат x1, x2...xm минимума функции;

F - массив чисел Фибоначчи;

n - количество чисел Фибоначчи;

j - переменная, принимающая значения 0,1...m-1.

В методе покоординатного спуска в качестве очередного направления поиска выбирают направление одной из координатных осей.
